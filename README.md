# Agentic-RAG
Adaptive RAG System – Intelligent Retrieval-Augmented Generation platform combining PDF document analysis and web search to deliver context-aware answers with verdicts: CORRECT (from document), INCORRECT (from web), and AMBIGUOUS (hybrid). Built with FastAPI, LangChain, FAISS, OpenAI GPT-4o-mini, and a professional ChatGPT-style frontend.
